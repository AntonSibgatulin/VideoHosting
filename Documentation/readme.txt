getmyprofile - получить профиль пользователя
	входные параметры: строка,порядок строгий ,пользователь должен быть авторизован
	Ответ json:
	setmyprofile
	1.json:
		1.myid - id пользователя
		2.name - (Имя)
		3.surname - Фомилия
		4.city - город
		5.email - почта
		6.date - дата рождения
		7.typeuser - число от 0-4 (4-Админ 0-Пользователь 2-Тестер 3-Модератор)
getprofile - получить профиль пользователя
	входные параметры :строка ,порядок строгий,Пользователь должен быть авторизован
	1.id - пользователя
	отевт : json
	1.json
		1.id - id пользователя
		2.name - (Имя)
		3.surname - Фомилия
		4.city - город or error access
		5.email - почта or error access
		6.date - дата рождения or error access
		7.typeuser - число от 0-4 (4-Админ 0-Пользователь 2-Тестер 3-Модератор)
		8.login - Логин к примеру @AntonSibgatulin 
		в 15 36 02.11.2020 error access не работает
		в 16 29 02.11.2020 error access работает
